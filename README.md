ระบบร้องเรียนปัญหาและแจ้งเตือนข่าวสารภายในมหาวิทยาลัย
University Issue Reporting and Emergency Notification System
ระบบนี้เป็นแพลตฟอร์มดิจิทัลส่วนกลางที่เชื่อมโยง นิสิต, บุคลากร, สโมสรคณะ, เจ้าหน้าที่มหาวิทยาลัย และผู้ดูแลระบบ เข้าด้วยกัน เพื่อเพิ่มประสิทธิภาพในการแก้ไขปัญหาและสื่อสารข้อมูลภายในมหาวิทยาลัย ระบบถูกออกแบบให้เป็นส่วนหนึ่งของระบบนิเวศดิจิทัล (Digital Ecosystem) โดยมีการเชื่อมต่อกับระบบฐานข้อมูลและระบบยืนยันตัวตนของมหาวิทยาลัย
